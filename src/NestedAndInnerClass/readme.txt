Вложенный и внутренний класс:
- nested(вложенный класс) классы всегда static
- inner(внутренний класс) классы всегда НЕ static

Вложенным классом называют класс, который является членом другого класса (находится в его теле).
Существует четыре базовых типа вложенных классов в Java:
    1 Static Nested (вложенный статический класс)
    2 Nested Inner (вложенный внутрений класс - всегда не static)
    3 Method Local Inner (вложенный класс в локальном методе)
    4 Anonymous.Anonymous Inner (анонимные классы)

    1 Static nested class
        - имеет доступ к private полям своего внешнего класса
        - имеет доступ к static членам внешнего класса, напрямую (без создания экземпляра)
        - НЕ имеет доступ к private НЕ static членам внешнего класса без его эксземпляра
        - НЕ имеет доступ к НЕ static членам внеш.класса без его экземпляра
        - nested class можно наследовать
